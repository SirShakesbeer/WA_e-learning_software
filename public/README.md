# WorkAdventure Map Starter Kit - Public Folder

In this directory you can put static files (audio files, PDFs, videos, images except tilesets...) that you want to be present to the players.

Die dialoge-Datei enthält momentan alle Dialoge. In der Zukunft könnten in diesem Ordner noch mehr Dialoge modularisiert eingefügt werden.

In `achievements` befinden sich die Popup achievements als html Dateien. Wenn diese in Skripten aufgerufen werden sollen müssen die Bezeichnungen ohne `.html` als argument in die Funktion `showAchievement` weitergegeben werden.
